# Image Sizes

This repository contains various images categorized into different directories based on their usage and dimensions. Below is a summary of the image sizes organized by directory:

## Fleet

- **Directory:** assets -> images -> fleet
- **Image Dimensions:** 410 pixels width * 270 pixels height

## Gallery

- **Directory:** assets -> images -> gallery
- **Image Dimensions:** 750 pixels width * 450 pixels height

## Specials

- **Directory:** assets -> images -> specials
- **Image Dimensions:** 500 pixels width * 500 pixels height

## Tours

- **Directory:** assets -> images -> tours -> pearl 1 & pearl 2
- **Image Dimensions:** 350 pixels width * 300 pixels height
